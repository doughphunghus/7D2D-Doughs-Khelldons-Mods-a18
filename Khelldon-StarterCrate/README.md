# Starter Crate
This modlet gives the player a crate of random gear to start the game with.

Details:
- Loot is mostly random but of the following types:
  1 armorMiningHelmet
  1 Melee weapon
  3 tools
  4 pieces of armor
  2 items of medicine
  2 canned food
